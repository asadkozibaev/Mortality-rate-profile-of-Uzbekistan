# Mortality Profile Analysis of Uzbekistan (1990–2019)

This project explores how the mortality profile of Uzbekistan has evolved between 1990 and 2019 using data from Our World in Data. Using Python (Pandas and Matplotlib), I analyzed leading causes of death, identified the top mortality drivers, visualized long-term trends, and compared cardiovascular mortality with France and Niger. The project demonstrates data cleaning, exploratory data analysis, visualization, and interpretation of findings for public policy applications.
